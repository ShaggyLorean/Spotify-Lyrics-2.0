# Spotify Lyrics 2.0 (Lorean Edition)

A KDE Plasma widget that displays synchronized lyrics from Spotify directly on your taskbar or desktop.

## Features
- **Synchronized Lyrics:** Fetches lyrics automatically from LRCLib.
- **Smart Search:** Finds lyrics even for "Feat." tracks or slight naming differences.
- **Visual Customization:** Change fonts, sizes, and highlight colors.
- **Performance:** Optimized ListView rendering for smooth scrolling.
- **Sync Offset:** Adjust timing if lyrics are too early/late.
- **Works with:** Spotify (Official & Spicetify).

## Installation

### From KDE Store
1. Right-click on your desktop or panel.
2. Select "Add Widgets..." > "Get New Widgets".
3. Search for "Spotify Lyrics".
4. Install.

### Manual Installation
1. Download the latest `.plasmoid` or `.tar.gz` release.
2. Run: `kpackagetool5 -i spotify-lyrics-2.0.plasmoid`
